# Plan Détaillé — 54 Cartes "Nexus Computation Moderne"

## 1. Guide Visuel & Style

### 1.1 Dimensions & Format
- **Ratio** : 3:4 (format poker standard, 88×63mm)
- **Résolution** : 2K (896×1200 px) via Gemini `gemini-3.1-flash-image-preview`
- **Format fichier** : PNG

### 1.2 Palette de couleurs

| Élément | Hex | Usage |
|---------|-----|-------|
| Fond principal | `#0a0e1a` | Noir graphite profond |
| Fond secondaire | `#111827` | Bleu nuit très sombre |
| CPU (Trèfle) | `#10b981` | Vert émeraude tech |
| Data/IA (Carreau) | `#06b6d4` | Cyan données |
| Sécurité (Coeur) | `#f43f5e` | Rouge bouclier / alerte |
| Quantique (Pique) | `#a855f7` | Violet quantique |
| Accent lumineux | `#f59e0b` | Ambre alerte / highlight |
| Texte principal | `#e2e8f0` | Blanc cassé |
| Texte secondaire | `#64748b` | Gris ardoise |

### 1.3 Style artistique
- **Style** : Digital art semi-réaliste, clean, haute lisibilité
- **Ambiance** : Data center futuriste, hologrammes, interfaces UI flottantes
- **Éclairage** : Rim light néon coloré selon la couleur, ambiance sombre
- **Fond** : Dégradé sombre avec motifs subtils (grilles, circuits, code binaire en filigrane)
- **Bordure** : Fine ligne lumineuse de la couleur de la couleur (2px glow)
- **Typographie** : Style monospace/terminal pour les indices (rang + symbole)

### 1.4 Structure d'une carte (composition)

```
┌─────────────────────┐
│ A  ◈               │  ← Index haut-gauche (rang + symbole)
│                     │
│                     │
│    [ILLUSTRATION    │  ← Zone centrale : illustration principale
│     PRINCIPALE]     │
│                     │
│                     │
│               ◈  A │  ← Index bas-droite (inversé pour double-head)
└─────────────────────┘
```

**Règles de composition :**
- Index dans les 2 coins opposés (haut-gauche, bas-droite)
- Zone safe : 10% de marge intérieure
- Illustration centrée, occupe ~70% de la hauteur
- Fond subtil avec motifs thématiques semi-transparents
- Les figures (V, D, R) sont **double-headed** (symétriques haut/bas)

## 4. Règles de Prompt Engineering pour la Cohérence

### 4.1 Template de base (à préfixer à chaque prompt)

```
You are generating a series of 54 playing cards for a custom deck themed 
"Nexus: Modern Computation" — a fusion of AI, cloud computing, quantum 
computing, and cybersecurity. 

STYLE RULES (apply to ALL cards):
- Dark graphite background (#0a0e1a to #111827 gradient)
- Clean, professional digital illustration style
- High contrast with neon accent lighting
- No real text on the card except the rank and suit symbol
- Playing card format, 3:4 aspect ratio
- Subtle circuit board trace patterns in the background
- Faint binary code watermark in the background
- Fine glowing border in the suit's accent color

Generate the following card:
```

### 4.2 Suffixe de qualité

```
Style: clean digital illustration, semi-realistic, high detail, 
professional playing card design. Dark moody lighting with colored 
neon rim light. Centered composition. No watermarks. No extra text.
```

### 4.3 Stratégie de cohérence
1. **Multi-turn chat** : Générer les cartes d'une même couleur dans une seule conversation
2. **Référence croisée** : Fournir les cartes déjà générées comme images de référence
3. **Batch par couleur** : CPU (13) → DATA (13) → SHIELD (13) → QUBIT (13) → Jokers (2) → Dos (1)
4. **Post-traitement Pillow** : Superposer les indices (rang + symbole) de manière uniforme

## 5. Post-traitement avec Pillow

Après génération par Gemini, chaque carte reçoit :
1. **Index haut-gauche** : rang (A, 2-10, V, D, R) + icône du symbole, police monospace, couleur de la couleur
2. **Index bas-droite** : même chose, rotation 180°
3. **Valeur Bridge** : petit nombre discret en bas-centre (1-53) pour usage dans le jeu Solitaire
4. **Bordure** : fine ligne glow (2px) de la couleur de la couleur
5. **Coins arrondis** : rayon 12px
